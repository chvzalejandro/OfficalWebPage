Wireless Utilities Monitoring System

Working alongside software and electrical engineers we created a mobile app that provided instant data of the utility usage. With this app you were able to see how much water you've used as well as the amount of energy consumed. Not only that, but the user was also able to see how much money they could save when using their AC/Heater.


License Plate Recognition

Using MATLAB I created a program that would automatically recognize characters from an image of a license plate and convert it into a text file. The program takes the image of the license plates and process the image through several filters. From this point the program then extracts each individual character from the filtered image and stores each individual character. Once all the induvial characters have been stored the program will then convert the characters onto a text file. The final step of the programs just double checks that the text file was converted correctly by running the program once more.


MicroMouse

Working with a software engineer, we created a robot that would traverse a maze autonomously and find the exit. Using the flood fill method, the robot would gradually navigate the maze while calculating the best route to the exit. The robot was also equipped with infrared sensors and steppers motors, using the sensors to not bump into walls and the steppers motors for more precise movement.
